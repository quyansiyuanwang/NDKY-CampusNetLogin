# 校园网自动登录工具

自动监控网络连接状态，断网时自动登录校园网。

## 功能特性

- ✅ 自动获取登录配置（queryString、RSA 公钥）
- ✅ 自动跟随所有 HTTP 302 重定向
- ✅ 定时检查外网连接状态
- ✅ 断网时自动登录
- ✅ 支持自定义检查间隔
- ✅ 使用 .env 文件管理敏感信息
- ✅ 模块化设计，代码简洁高效

## 快速开始

### 1. 配置账号信息

编辑 `.env` 文件：

```env
# 用户名
USERNAME=你的学号

# 密码
PASSWORD=你的密码

# 服务器地址
BASE_URL=http://10.88.108.101

# 检查间隔（秒），默认 5 秒
CHECK_INTERVAL=5
```

### 2. 运行程序

```bash
node main.js
```

程序会：

1. 自动获取登录所需的配置信息
2. 每隔 N 秒检查一次外网连接
3. 如果无法访问外网，自动执行登录
4. 登录成功后继续监控

### 3. 停止程序

按 `Ctrl+C` 停止运行。

## 工作原理

1. **配置获取**：访问校园网登录页面，自动提取 queryString 和 RSA 公钥
2. **网络检查**：定时访问 Microsoft 连接测试服务检查外网连接
3. **自动登录**：检测到断网时，使用 RSA 加密密码并提交登录请求
4. **循环监控**：持续监控网络状态，确保始终在线

## 其他工具

### get-querystring.js

手动获取登录配置信息：

```bash
node get-querystring.js
```

该工具会：

- 自动跟随所有重定向
- 提取 queryString 参数
- 调用 pageInfo API 获取 RSA 公钥
- 生成可直接使用的配置代码

## 技术特点

### 模块化设计

- **utils.js**：共享工具函数库
  - HTTP 请求封装（GET/POST）
  - URL 处理工具
  - 重定向跟随
  - Portal API 封装

### 高效实现

- 使用 Buffer 处理 HTTP 响应，避免字符串拼接
- 配置缓存，避免重复获取
- 统一的常量管理
- 无代码重复

### 安全性

- 密码使用 RSA 加密传输
- 敏感信息存储在 .env 文件
- .gitignore 保护配置文件

## 注意事项

1. `.env` 文件包含敏感信息，请勿提交到版本控制系统
2. 检查间隔不要设置太短，避免频繁请求
3. 确保已连接到校园网物理网络
4. drop 文件夹包含开发过程中的废弃文件，可以安全删除

## 故障排除

### 无法获取配置

- 确保已连接到校园网
- 检查服务器地址是否正确
- 尝试在浏览器中访问登录页面

### 登录失败

- 检查用户名和密码是否正确
- 查看错误信息，可能是账号欠费或其他问题
- 尝试手动在浏览器中登录测试

### 网络检查失败

- 检查是否能访问外网
- 防火墙可能阻止了连接测试
